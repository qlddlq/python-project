# 🚢 Titanic Survival Analysis

## 📌 Overview
This project analyzes the Titanic dataset to identify key factors that influenced passenger survival.

It demonstrates core data analysis skills including data cleaning, visualization, and feature engineering.

---

## 🎯 Objectives
- Understand passenger demographics
- Analyze survival patterns
- Create new features for better insights
- Communicate findings clearly

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## ▶️ How to Run
1. Clone the repository:
```bash
git clone https://github.com/yourusername/titanic-analysis.git
```
2. Navigate to the project:
```bash
cd titanic-analysis
```
3. Install dependencies:
```bash
pip install -r requirements.txt
```
4. Run Jupyter Notebook:
```bash
jupyter notebook
```
5. Open:
```bash
notebook/analysis.ipynb
```
