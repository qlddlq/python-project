# 🚢 Titanic Survival Analysis

## 📌 Overview
This project analyzes the Titanic dataset to identify key factors that influenced passenger survival.

It demonstrates core data analysis skills including data cleaning, visualization, and feature engineering.

---

## 🎯 Objectives
- Understand passenger demographics
- Analyze survival patterns
- Create new features for better insights
- Communicate findings clearly

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
