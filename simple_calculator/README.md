# 🧮 CLI Calculator (Python)

## 📌 Description
This is a simple command-line calculator built using Python.  
It performs basic arithmetic operations and demonstrates core programming concepts such as functions, conditionals, and user input handling.

---

## 🚀 Features
- Addition, subtraction, multiplication, division
- Continuous calculation loop (until user exits)
- Error handling (e.g., division by zero)
- Clean and simple command-line interface

---

## 🛠️ Technologies Used
- Python 3

---

## ▶️ How to Run

1. Clone this repository:
```bash
git clone https://github.com/qock1210-maker/python-projects.git
```
2. Navigate to the calculator folder:
```bash
cd python-projects/simple_calculator
```
3. Run the program:
```bash
python calculator.py
```
